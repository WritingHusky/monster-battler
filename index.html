Welcome to my portfolio project showcasing my skills in backend development! Here, I present to you a simplified version of the beloved Pokémon Showdown. This project comprises three integral parts:
  1. Monster Battling System: Implemented in Java, this handles all the intricate logic behind engaging battles between digital creatures. From move selection to HP management, experience the essence of Pokémon battles in this meticulously crafted system.
  2. Spring-Boot REST API: Delve into the heart of the action with a robust API built on Spring Boot. This component orchestrates the interactions within battles, facilitating seamless communication between users and AI opponents.
  3. React Frontend: Although aesthetics take a backseat, the React frontend (in TypeScript) provides a functional interface for users to engage with the battle system. Experience the thrill of strategic gameplay through a simple yet effective user interface.
Whether you're a fellow developer, an avid Pokémon fan, or simply curious about backend development, explore this project to witness firsthand the culmination of my skills and passion. Let the battles begin!
